# 🎓 EduMate AI – Smart AI Learning Assistant

## 🌟 Overview

**EduMate AI** is an AI-powered educational platform developed to improve the learning experience through intelligent assistance and productivity tools. The project integrates modern AI technologies with web development to provide students with academic support, study management, and interactive learning features.

The platform helps students stay organized, access educational assistance instantly, and improve learning efficiency through a user-friendly interface.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Zehara-K/EdumateAI_Project.git
```

```bash
cd EdumateAI_Project
```

### Install Backend Dependencies
```bash
cd Backend
npm install
```
### Install Frontend Dependencies
```bash
cd Frontend
npm install
```
### Configure Environment Variables
Create a `.env` file inside the Backend folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
GEMINI_API_KEY=your_api_key
```
### Start Backend Server
```bash
node server.js
```
### Start Frontend

```bash
npm run dev
```
